= Bubble Parameter Tuning Quick Reference
:toc:

== Motion Parameters

=== Rise Speed
* *Current*: 35.0 units/sec
* *Effect*: How fast bubbles ascend after release
* *Tune up*: Faster, more energetic bubbles
* *Tune down*: Slower, more languid rise
* *Range*: 20.0 - 50.0

=== Drift Speed
* *Current*: 3.0 units/sec
* *Effect*: Lateral (X/Z) movement speed
* *Tune up*: More wandering path
* *Tune down*: Straighter vertical rise
* *Range*: 1.0 - 5.0

=== Wobble Frequency
* *Current*: 0.6 Hz
* *Effect*: How fast the drift pattern oscillates
* *Tune up*: Faster side-to-side motion
* *Tune down*: Slower, gentler sway
* *Range*: 0.3 - 1.0

=== Wobble Amplitude
* *Current*: 0.12 (12% of diameter)
* *Effect*: How much the bubble sways
* *Tune up*: More dramatic path variation
* *Tune down*: Subtler wobble
* *Range*: 0.05 - 0.2

== Visual Parameters

=== Opacity
* *Current*: 0.35 (35% opaque)
* *Effect*: How see-through bubbles are
* *Tune up*: More visible/solid bubbles
* *Tune down*: More transparent, ethereal
* *Range*: 0.2 - 0.6

=== Roughness
* *Current*: 0.15 (quite shiny)
* *Effect*: Surface shininess vs matte
* *Tune up*: More matte/frosted appearance
* *Tune down*: Shinier, more reflective
* *Range*: 0.1 - 0.5

=== Color
* *Current*: [0.7, 0.9, 1.0] (light cyan)
* *Effect*: Bubble tint color
* *Cyan*: [0.7, 0.9, 1.0] - classic underwater
* *Blue*: [0.6, 0.8, 1.0] - deeper water
* *White*: [1.0, 1.0, 1.0] - pure bubbles
* *Green*: [0.7, 1.0, 0.9] - murky water

=== Diameter Multiplier
* *Current*: 1.1 (10% larger than deformation)
* *Effect*: Bubble size relative to cylinder deformation
* *Tune up*: Larger bubbles (risk of clipping)
* *Tune down*: Smaller, more contained bubbles
* *Range*: 0.8 - 1.2
* *WARNING*: >1.15 may clip through walls with multiple Tendroids

== Pop Timing Parameters

=== Min Pop Time
* *Current*: 21.0 seconds
* *Effect*: Minimum bubble lifetime before pop
* *Tune up*: Bubbles live longer minimum
* *Tune down*: Earlier pops possible
* *Range*: 10.0 - 30.0

=== Max Pop Time
* *Current*: 39.0 seconds
* *Effect*: Maximum bubble lifetime before pop
* *Tune up*: Some bubbles live much longer
* *Tune down*: All pops happen sooner
* *Range*: 25.0 - 60.0

*Relationship*: Recommended 30% variation (e.g., 21-39s around 30s base)

== Pop Effect Parameters

=== Particles Per Pop
* *Current*: 7 droplets
* *Effect*: Number of spray particles per pop
* *Tune up*: More dramatic burst (costs performance)
* *Tune down*: Subtler pop effect
* *Range*: 4 - 10
* *Performance*: 50 max total particles

=== Particle Speed
* *Current*: 18.0 units/sec
* *Effect*: Initial velocity of spray droplets
* *Tune up*: More explosive spray
* *Tune down*: Gentler pop
* *Range*: 10.0 - 25.0

=== Particle Lifetime
* *Current*: 0.6 seconds
* *Effect*: How long droplets persist
* *Tune up*: Longer fade (more visible)
* *Tune down*: Quicker disappearance
* *Range*: 0.3 - 1.0
* *Performance*: Longer = more particles accumulate

=== Particle Spread
* *Current*: 50.0 degrees
* *Effect*: Cone angle of spray pattern
* *Tune up*: Wider spray (hemisphere)
* *Tune down*: Narrower, more upward spray
* *Range*: 30.0 - 90.0

=== Particle Size
* *Current*: 0.4 world units
* *Effect*: Visual size of droplets
* *Tune up*: More visible droplets
* *Tune down*: Finer mist effect
* *Range*: 0.2 - 0.8

== Quick Tuning Scenarios

=== More Dynamic Underwater Scene
[source,json]
----
"rise_speed": 40.0,        // Faster ascent
"drift_speed": 4.0,        // More wandering
"wobble_frequency": 0.8,   // More active motion
----

=== Calmer, More Serene
[source,json]
----
"rise_speed": 28.0,        // Slower rise
"drift_speed": 2.0,        // Less drift
"wobble_frequency": 0.4,   // Gentle sway
----

=== More Dramatic Pops
[source,json]
----
"particles_per_pop": 9,    // More droplets
"particle_speed": 22.0,    // Faster spray
"particle_lifetime": 0.8,  // Longer visible
----

=== Subtler Pops
[source,json]
----
"particles_per_pop": 5,    // Fewer droplets
"particle_speed": 15.0,    // Gentler spray
"particle_lifetime": 0.4,  // Quick fade
----

=== Maximum Performance
[source,json]
----
"max_bubbles_per_tendroid": 3,  // Fewer bubbles
"particles_per_pop": 5,         // Fewer particles
"particle_lifetime": 0.4,       // Quick cleanup
"max_particles": 30             // Lower limit
----

== Testing Workflow

. Edit `config/tendroids_config.json`
. Save changes
. Reload Omniverse scene or restart extension
. Observe bubbles for 30+ seconds to see pops
. Adjust parameters and repeat

== Common Issues

=== Bubbles Clipping Through Walls
* *Cause*: `diameter_multiplier` too high
* *Fix*: Reduce to 1.0 or 1.05
* *Current safe value*: 1.1

=== Bubbles Too Transparent
* *Cause*: `opacity` too low
* *Fix*: Increase to 0.4 - 0.5

=== Pops Not Visible
* *Cause*: Particles too small or fast lifetime
* *Fix*: Increase `particle_size` to 0.5-0.6
* *Fix*: Increase `particle_lifetime` to 0.8-1.0

=== Motion Too Erratic
* *Cause*: High wobble_frequency or drift_speed
* *Fix*: Reduce both to 0.4 and 2.0

=== Not Enough Bubbles
* *Cause*: Low `max_bubbles_per_tendroid`
* *Fix*: Increase to 6-8
* *Warning*: May impact performance
