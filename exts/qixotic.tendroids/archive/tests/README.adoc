== Using the UI Instead

The Hide Until Clear feature is now integrated into the Tendroids Controls UI:
1. Open the Tendroids Controls panel
2. Expand "Bubble Controls" section
3. Check/uncheck "Hide Until Clear" checkbox

This is the recommended way to control the feature.
