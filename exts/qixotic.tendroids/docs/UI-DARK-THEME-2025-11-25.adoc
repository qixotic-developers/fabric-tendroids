= UI Improvements - Dark Theme & Scrolling
:date: 2025-11-25

== Changes Made

=== 1. Much Darker Control Backgrounds

**Slider controls:** Changed from `0xFF333333` → `0xFF0A0A0A` (almost black)
- Creates strong contrast with container background (`0xFF23211F`)
- Matches Property window style from screenshot
- Applied to all int/float sliders

**Updated in:** `slider_row.py`

=== 2. Container Backgrounds

**Container color:** `0xFF23211F` (medium dark gray)
- Applied to all CollapsableFrame contents:
  * Spawn Settings
  * Wave Motion
  * Bubble Settings
  * Actions

**Padding:** 4px spacers at top/bottom of each section

=== 3. Scrollable Content

**ScrollingFrame wrapper:**
- Horizontal scrolling: NEVER
- Vertical scrolling: AS_NEEDED
- Background: `0xFF23211F`
- Bottom spacer (20px) for clearance

**Benefits:**
- Content adapts to any window height
- Smooth scrolling when needed
- All controls accessible regardless of window size

=== 4. Window Configuration

**Window title:** "Tendroid Manager"
**Size:** 320×600 (scrollable, so can be smaller)
**Dock preference:** RIGHT_BOTTOM (right panel)
**Visible by default:** Yes

== Console Customization

**Status:** Not implemented

The Kit SDK console is a core system component with fixed formatting that cannot be customized from extensions. Options:

1. **Use as-is** - Accept default console formatting
2. **Custom log window** - Create filtered tendroid-only log with custom styling

Let me know if you want option #2.
