= Fabric Tendroids Documentation
:page-layout: home

Documentation for the Fabric Tendroids Omniverse extension.

== Overview

Fabric Tendroids is an NVIDIA Omniverse extension that creates interactive 3D simulations featuring creature-tendroid interactions with GPU-accelerated physics.

== Key Features

* *Real-time Physics* - GPU-accelerated collision detection and response
* *Interactive Creatures* - Cylindrical creatures with defensive behaviors
* *Tendroid System* - Proximity-based physics interactions
* *Warp Integration* - NVIDIA Warp for GPU computing
* *USD/Fabric* - Universal Scene Description integration

* NVIDIA RTX GPU (Turing or newer)

* 16GB+ system RAM
