== Overview

The wave motion system simulates gentle ocean currents affecting Tendroids and bubbles, creating a realistic underwater environment with synchronized swaying motion.

== Architecture

=== Components

* **WaveController**: Global wave state manager
* **WaveConfig**: Configuration parameters  
* **AnimationController**: Integrates wave updates with animation loop
* **Tendroid**: Applies graduated displacement from base to tip

=== Motion Characteristics

* **Base behavior**: Anchored with zero movement
* **Tip behavior**: Maximum displacement following wave pattern
* **Body segments**: Smooth cubic interpolation from base to tip
* **Phase offset**: Slight variation between individual Tendroids
