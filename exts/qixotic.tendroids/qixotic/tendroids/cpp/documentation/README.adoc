== Integration

Once USD is enabled, replace Python implementation:

[source,python]
----
# OLD: Python tuple conversion (35ms)
from .fabric_batch_updater import FabricBatchMeshUpdater

# NEW: C++ direct pointer access (<1ms)
from .cpp_fabric_updater import CPPFabricBatchUpdater
----

See `warp_test/cpp_fabric_updater.py` for drop-in replacement wrapper.
