:toc:
:toclevels: 2

== Overview

Added real-time bubble parameter controls to the Tendroid UI panel.

== UI Controls

=== Geometry Section
* *Diameter Multiplier* (0.1 - 2.0)
** Controls bubble size relative to deformation
** Default: 1.2 (from JSON)
** Recommended: 1.25 for single Tendroid, 0.5-0.8 for multiple

=== Pop Timing Section
* *Min Pop Time* (5 - 30 seconds)
** Minimum time before bubble pops
** Default: 10.0s
** Ensures bubbles stay visible

* *Max Pop Time* (5 - 40 seconds)
** Maximum time before bubble pops
** Default: 25.0s
** Creates varied pop timing

=== Physics Section
* *Rise Speed* (10 - 50 units/sec)
** How fast bubbles float upward
** Default: 30.0
** Higher = faster ascent

== Configuration Hierarchy

1. *JSON Config* (tendroids_config.json line 75)
   - Source of truth on startup
   - Edit this for permanent changes

2. *UI Sliders*
   - Real-time adjustments during session
   - Changes NOT saved to JSON
   - Useful for tuning values

3. *Python Defaults* (bubble_config.py)
   - Fallback only if JSON missing values
   - Should match JSON to avoid confusion

== Usage

1. Load extension
2. Controls appear in "Bubble Controls" collapsable section
3. Adjust sliders to tune behavior in real-time
4. Note values you like
5. Edit JSON to make changes permanent
