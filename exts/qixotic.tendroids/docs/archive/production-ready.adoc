== Development Setup

=== Prerequisites
* USD Composer 2024.1.0+
* Visual Studio 2022
* Python 3.10
* pybind11

=== Build C++ Extension
[source,bash]
----
# In CLion
1. File → Open → cpp/
2. Build → Build Project (Ctrl+F9)
3. Copy fast_mesh_updater.pyd to qixotic/tendroids/

# Or use build script
cd cpp
build_simple.bat
----

=== Run Tests
[source]
----
1. Start USD Composer
2. Extensions → Qixotic Tendroids → Performance Tests
3. Click "Run Test"
4. Watch console and HUD for FPS
----
