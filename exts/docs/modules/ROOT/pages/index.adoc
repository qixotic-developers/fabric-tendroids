= Geo-Sub OptiX Documentation
:page-layout: home

Comprehensive documentation for the geo-sub-optix ray tracing application.

== Overview

This project demonstrates real-time geometric subdivision using NVIDIA OptiX 9.0 
and CUDA 13.0 for hardware-accelerated ray tracing of icosahedron meshes.

== Key Features

* *Hardware Ray Tracing* - NVIDIA OptiX 9.0 with RT cores
* *CUDA Integration* - GPU-accelerated subdivision algorithms
* *Multiple Subdivision Strategies* - Five distinct algorithms
* *Real-time Interactive Rendering* - GLFW windowing with OpenGL display
* *Modern C++20* - Latest language features
