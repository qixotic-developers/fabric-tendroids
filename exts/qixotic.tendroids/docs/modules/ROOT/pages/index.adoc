== Welcome

Welcome to the Tendroids project documentation.

This project implements tendroid-creature interactions in Nvidia Omniverse using:

* Python 3.x
* USD (Universal Scene Description)
* Warp GPU computing
* Omniverse Fabric

== Project Overview

The Tendroids project features:

* Cylindrical creatures with proximity-based physics
* Tendroid detection and avoidance behaviors
* Defensive mechanisms including bending and shock feedback
* GPU-accelerated collision detection
